What GMPE Rewards

Traditional platforms reward labor, attention, and content. GMPE v2.0 rewards observation, experience, verification, and continuity.

Work Layer

The current GMPE system remains active as a human execution network for AI testing, product feedback, surveys, data collection, and annotation tasks.

Trust Layer

ACP-ATP turns behavior into portable trust: Observation Score, Execution Score, Reliability Score, and Consistency Score.

Experience Layer

The Life Capsule Program preserves photos, videos, audio, journals, emotion tags, and agent analysis as encrypted experience assets.

Experience Score UI

The score measures continuity, authenticity, contribution, and quality. It is designed to make verified human experience accumulative instead of disposable.

Data Ownership Rule

Original creators always own original data. GMPE does not buy user data. Contributors grant licenses, not ownership transfer. Encrypted capsule content can be stored through IPFS while access rules remain explicit.
